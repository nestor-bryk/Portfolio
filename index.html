Skills

What do I use at work?
HTML
HTML is a programming language used to create web pages. With the help of HTML, you can mark up text, images, videos and other elements of a web page.
CSS
CSS is a programming language used to design web pages created using HTML. With CSS, you can define the appearance of a web page, including colors, fonts, placement of elements, and more.
Java Script
JavaScript is a programming language used to create dynamic web pages and user interaction. JavaScript allows you to add functionality to web pages that cannot be accomplished using just HTML and CSS.
Jquery
jQuery is a JavaScript library that helps simplify interactions with HTML documents, including page navigation, event handling, animations, working with forms, and more.
React
React is a JavaScript library for building user interfaces (UI) developed by Facebook. React allows developers to create dynamic web pages using components that can be reused across different parts of the project.
Redux
Redux helps manage complex application states by storing them in a centralized "store". The state of the application is stored as an object containing all the necessary data to display the user interface.
Bootstrap
Bootstrap is an open framework for developing web applications and sites. It contains a set of ready-made components that allow you to quickly and easily create a stylish and responsive user interface.
Git
Git is a distributed version control system used to track changes in software code and collaborate on projects. It allows you to save copies of the code on different computers and exchange them with other developers over the Internet.
Sass
It is a programming language used to write CSS styles. It allows developers to create and maintain styles for websites more conveniently and quickly.
Webpack
Webpack is a tool for collecting and managing web application dependencies. It allows developers to break their code into modules and load them in an optimized order, which allows faster loading of web applications and reduces their weight.
Less
Less is designed to make it easier to write and maintain styles for web applications and allows you to use variables, functions, operators, and other programming constructs in CSS.
